This is my first attempt at writing a graphing calculator. Currently, it can handle addition (+), subtraction (-), multiplication (*), division (/) and exponentiation (^)

It's still missing a few features:

  • Unary minus was implemented as an afterthought and can be improved.
  • Functions like absolute value, trigonometric functions, square root, etc have not yet been implemented.
  • Proper scaling needs to be implemented for drawing the graph.
  • Need ability to clear the graph without refreshing the page.
  • Multple lines on the graph canvas could be different colors.

Feel free to suggest improvements.
